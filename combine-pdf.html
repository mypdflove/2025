How to Use PDF Combiner

1
Select PDF Files
Click the selection box or drag and drop multiple PDF files to combine.
2
Arrange Files (Optional)
Use the up and down arrows to rearrange the order of PDF files.
3
Combine PDFs
Click the "Combine PDFs" button to merge all selected files into a single PDF.
4
Download
Your combined PDF will automatically download when the process is complete.
Pro Tip

You can select multiple PDF files at once by holding Ctrl (or Cmd on Mac) while selecting files.
